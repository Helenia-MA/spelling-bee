# Spelling 🐝ee
I've developed a browser word game built with react.
It entails forming as many words as possible from a set of random letters before the timer or the chances run out.

## Live Demo
https://spelling-bee-cwas.onrender.com

## How to play
1. input your name the click 'play' to start the game
2. you're given 5 random letters
3. form as many valid words as possible using those letters (you can use each tile only once)
4. each word's point = its length
5. you get 3 hints per round
6. 0.5 points are deducted every 2 minutes!

## Tech Stack
- React + Vite
- Deployed on Render

## Running it locally
```bash
npm install
npm run dev
```
Then open http://localhost:5175
